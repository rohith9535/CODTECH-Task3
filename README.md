## Personal Portfolio Website

Check The Website:-  https://rohith9535.github.io/CODTECH-Task3/

This project is a responsive personal portfolio website developed to present my profile, technical skills, selected projects, and contact details in a clean and professional layout. The website is designed with a modern user interface and focuses on readability, accessibility, and responsive behavior across different screen sizes.

The portfolio includes sections such as Home, About Me, Projects, Skills, and Contact, making it easy to showcase frontend development abilities and personal branding in a single-page format. It is built using basic web technologies and demonstrates practical knowledge of layout design, styling, and responsive web development.

## Features

- Responsive single-page portfolio design
- Clean and modern user interface
- About, Projects, Skills, and Contact sections
- Mobile-friendly navigation menu
- Simple and well-structured HTML and CSS code

## Technologies Used

- HTML5
- CSS3

## Project Purpose

The purpose of this project is to create a professional portfolio website that highlights my frontend development skills and provides a strong online presence for personal and career growth.
